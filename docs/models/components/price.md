# Price

An object with the properties and price of a single message

## Example Usage

```typescript
import { Price } from "@gsmservice-pl/messaging-sdk-typescript/models/components";

let value: Price = {
  error: null,
  cid: "custom-id-A44445T",
  type: 1,
  recipient: "+48999999999",
  sender: "Bramka SMS",
  parts: 1,
  unicode: true,
  flash: false,
  price: 0.16,
};
```
