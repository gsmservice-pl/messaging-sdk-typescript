# SmsMessage

An object with a new SMS message properties

## Example Usage

```typescript
import { SmsMessage } from "@gsmservice-pl/messaging-sdk-typescript/models/components";

let value: SmsMessage = {
  recipients: "+48999999999",
  message: "To jest treść wiadomości",
  unicode: true,
};
```
